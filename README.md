Порция – новое видение на сервис для учета питания.
Ключевые особенности и идея заключаются в том, что заказы из «Яндекс.
Лавка» автоматически отображаются в приложении, позволяя
пользователю:
1. Рассчитать оптимальный рацион на основе данных пользователя
2. Отслеживать энергетическую ценность и питательные элементы корзины, а
также количество рационов, на которых её хватит
3. Отслеживать заканчивающиеся продукты и/или продукты с истекающим
сроком годности
4. Легкий доступ к заказанным продуктам для записи в рацион
5. Информативная статистика по соблюдению средне-суточного рациона и
расходам на продукты, в разрезе категорий
6. Разделение заказов по магазинам, кафе, ресторанам (Для интеграции с «Яндекс.
Еда») и адресам доставки
7. Вести синхронизированный учет в рамках семейного аккаунта
